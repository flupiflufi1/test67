# Сборка из исходников
## Windows
Сначала,
1. Нажмите на Win+R
2. Введите cmd

Затем скачайте или склонируйте репозиторий. (Если у вас установлен git, то введите команду `git clone https://github.com/LemoLev/tgwsp-android-rewrite.git`, а если нет, то скачайте с главной страницы через кнопку "Code"->"Download ZIP" и разархивируйте его куда-то)

После этого, в cmd, перейдите в папку с репозиторием
1. Если вы клонировали через git, то просто `cd tgwsp-android-rewrite`
2. Если вы скачивали через сайт, то вам нужно найти то место, где лежит папка репозитория и ввести `cd tgwsp-android-rewrite`.

Для начала сборки, установите джаву с [java.com](https://www.java.com/ru/download/)
После установки, удостоверьтесь, что вы сделали всё правильно, сделав:
1. Введите `echo %JAVA_HOME%`
2. Если вывелось что-то похожее на путь, то вы всё сделали правильно и можете переходить к самой сборке. (НЕ ЗАКРЫВАЙТЕ CMD!)

В открытом вами cmd, вводите "gradlew build".
